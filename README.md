# 基于C++和Qt的共享账本设计

### 一、简要介绍

本项目实现了一个带有图形界面的共享账本。项目基于Qt creator进行开发，Qt框架版本6.5.1，编译环境MinGW 11.2.0。

### 二 使用方式

1.可下载源码使用Qt creator进行构建后运行

2.若无Qt环境，可以直接下载Releases版本运行，无需任何依赖。若需要测试用例，可将文件夹内的测试数据文件夹内的文件移动至可执行程序目录中。 若不移动，则系统无初始数据，需要输入初始密码。
